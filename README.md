# SafeSWE-Lite

A lightweight coding agent harness inspired by SWE-agent, with layered
deterministic guardrails, HITL approval, and mock-LLM verified feedback loops.

This repository is part of the AI4SE course project at NJU. The project is in
the requirements and planning phase: implementation code will start only after
`SPEC.md`, `PLAN.md`, and the required cold-start validation are complete.

## Current Planning Materials

- `PROJECT_REQUIREMENTS_RESEARCH.md`: external course requirements, project
  positioning, and initial scope decisions.
- `reference_projects_analysis.md`: notes on SWE-agent, mini-SWE-agent,
  AutoCodeRover, Agentless, OpenHands, and Aider.
- `GIT_CICD_DEVELOPMENT_WORKFLOW.md`: planned Git, PR, CI/CD, deployment, and
  course-evidence workflow.
